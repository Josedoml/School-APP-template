⚠️  IMPORTANT NOTES

1. SECRET_KEY: Change the key above to a secure random value

2. DATABASE: Use PostgreSQL in production (not SQLite)
   Railway provides PostgreSQL plugin

3. HTTPS: Railway automatically provides SSL certificates

4. Rate Limiting: Consider adding protection against brute force attacks

5. CORS: Change allow_origins=["*"] if you know frontend domain

🆘 TROUBLESHOOTING
────────────────────────────────────────────────────────────────────────────
App won't start?
  → Check Railway logs: railway logs
  → Verify environment variables are set
  → Ensure database connection string is correct

Auth errors?
  → Clear browser localStorage
  → Re-login with correct credentials
  → Check SECRET_KEY is consistent

Can't push to GitHub?
  → Use personal access token (not password)
  → Verify repository is created at GitHub
  → Check git remote: git remote -v
